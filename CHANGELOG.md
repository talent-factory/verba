## [0.1.1] - 2026-02-24

### Fixed

- Windows audio recording: robust device detection supporting both ffmpeg v7 (section-based) and v8+ (inline `(audio)`) output formats
- PowerShell fallback (`Win32_SoundDevice`) when ffmpeg finds no audio devices
- Whisper silence/hallucination detection (recordings with only dots/ellipsis are rejected with a helpful message)
- `install:local` npm script now works cross-platform (no more shell path issues on Cygwin)
- `make dev` workflow compatible with Windows/Cygwin

### Added

- `verba.audioDevice` setting for manual microphone selection on Windows
- Quick Pick dialog for microphone selection on first Windows recording
- `dictation.selectAudioDevice` command to change the microphone at any time
- Windows-specific ffmpeg search paths (Chocolatey, Scoop, WinGet, Program Files)
- Debug logging throughout the transcription and cleanup pipeline
